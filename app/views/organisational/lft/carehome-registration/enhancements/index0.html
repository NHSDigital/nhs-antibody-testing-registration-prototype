Your organisation type will determine which types of coronavirus (COVID-19) test kits you can register and if you can use this service to report rapid lateral flow test results.

What type of tests can you register?

There are 2 types of test kit:

  • a PCR test (swab test sent to a lab)
  • a lateral flow test (result is displayed on the test kit)

If you're registering lots of tests, use the correct spreadsheet to save time (if you dont have this, you can download it during the registration process).

Care homes can you this service to:

  • Register PCR tests
  • Register lateral flow tests
  • Report lateral flow test results

Schools and prisons can you this service to:

  • Register PCR tests
  • Register lateral flow tests

All organisations can you this service to:

  • Register PCR tests

Before you start

  • read the online guidance pack and your printed instructions
  • read the online spreadsheet guidance pack, using the spreadsheet will save you time when registering multiple people
  • if you need to, book your courier collection here
  • if you need to, order test kits here
  • take tests on the same day you're returning the kits

Get consent

Make sure people getting tested have freely given you consent to:

  • register them
  • get their results sent to the contact details you enter

What you'll need

  • your 8 digit organisation number
  • if you're registering lots of tests, use the correct spreadsheet (if you don't have this, you can download it during this registration process)
  • you'll need to record the details of staff and non-staff on separate spreadsheets
  • the barcode for each test
  • the email address (and ideally mobile number) to get the test results

Alternate ways to register

  • If your organisation type is eliable to register lateral flow tests, you can ask your organisation member's to use the self-report service to register their own tests
  • If a person within your care has been asked to be tested before visiting a hospital you can register a patient's PCR home test kit before their hospital visit
  • Pooled tests or tests done at a lateral flow test site must be register another way

How we use personal data

To find out how we use people’s data when you register them for a coronavirus test, read the coronavirus privacy note.
