What type of tests can you register?

There are 2 types of test kit:

  • a PCR test (swab test sent to a lab)
  • a lateral flow test (result is displayed on the test kit)

Your organisation type will determine whether you are able to register and report lateral flow test results.

All organisations can use this service to:

  • Register PCR tests

Care homes can use this service to:

  • Register PCR tests
  • Register lateral flow tests
  • Report lateral flow test results

Schools and prisons can use this service to:

  • Register PCR tests
  • Register lateral flow tests

Before you start

  • read the online guidance pack and your printed instructions
  • read the online spreadsheet guidance pack - if you’re registering lots of tests, using the correct spreadsheet will save you time
  • if you need to, book your courier collection
  • take the tests on the same day your courier comes to collect them

Get consent

Make sure people getting tested have freely given you consent to:

  • register them
  • get their results sent to the contact details you enter

What you'll need

  • your 8 digit organisation number
  • you’ll need to record the details of staff and non-staff on separate spreadsheets (if you don’t have the correct spreadsheet, you can download it during this registration process)
  • the barcode for each test
  • the email address (and ideally mobile number) to get the test results

How we use personal data

To find out how we use people’s data when you register them for a coronavirus test, read the coronavirus privacy note.
