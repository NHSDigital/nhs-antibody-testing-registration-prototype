Use this service for patients who have a hospital appointment for a procedure.

When to order a home test

Order tests for your patients 7 to 9 days before their hospital procedure. This allows enough time for them to take and return their test kit.

There are 2 ways you can order a test:

  • Use our spreadsheet to order tests for up to 50 patients at a time
  • Add each person's details one by one using our online form

Check what you need

You'll need the following patient details to request a test:

  • name
  • date of birth
  • gender
  • check if they have coronavirus symptoms
  • mobile phone number
  • NHS number
  • delivery address
  • date of procedure
